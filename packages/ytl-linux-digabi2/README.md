# ytl-linux-digabi2

This metapackage installs everything required to run the Abitti 2 server application (Docker, Naksu 2).

**IMPORTANT:** This package only works in native YTL-Linux installations (not WSL). For WSL, you must install [ytl-linux-digabi2-wsl](../ytl-linux-digabi2-wsl) instead.

To build the deb package, run:

```bash
just deb
```
